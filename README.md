# 🏢 Company Management System

A web-based Company Management System built with **ASP.NET Core MVC**, 
designed to streamline and manage company operations efficiently.

---

## 🚀 Features

- 👥 Employee management (Add, Edit, Delete, View)
- 🏗️ Department management
- 📋 Role-based access control
- 🔍 Search and filter functionality
- 📊 Dashboard with company overview

---

## 🛠️ Tech Stack

| Layer      | Technology              |
|------------|--------------------------|
| Framework  | ASP.NET Core MVC         |
| Language   | C#                       |
| Database   | SQL Server               |
| ORM        | Entity Framework Core    |
| Frontend   | Razor Views, Bootstrap   |
